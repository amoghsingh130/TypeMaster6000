# TypeMaster6000

TypeMaster6000 is a typing speed test application designed to help users improve their typing skills. The app provides real-time feedback on typing speed (WPM), accuracy, and progress over time. With a user-friendly interface and engaging features, TypeMaster6000 makes typing practice fun and effective.

## Features

- **Typing Tests**: Randomized text prompts to challenge your typing skills.
- **Real-Time Feedback**: Displays WPM and accuracy as you type.
- **Progress Tracking**: Visual representation of your typing performance over time.
- **Light and Dark Themes**: Aesthetic options to suit user preferences.
- **User-Friendly Interface**: Clean design with intuitive navigation.
- **Settings Menu**: Options to toggle themes and view app information.

## Technologies Used

- **Kotlin**: The primary programming language for Android development.
- **Android SDK**: The software development kit for building Android applications.
- **Material Design**: Google's design language for creating visually appealing user interfaces.
- **MPAndroidChart**: A library for displaying charts and graphs to visualize performance data.

## Installation

To set up the project locally, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/TypeMaster6000.git
   cd TypeMaster6000
   ```

2. **Open the Project**:
   Open the project in Android Studio.

3. **Sync Gradle**:
   Click on "Sync Project with Gradle Files" in Android Studio to ensure all dependencies are correctly set up.

4. **Run the Application**:
   Connect an Android device or start an emulator, then click the "Run" button in Android Studio.

## Usage

1. **Start the App**: Launch the TypeMaster6000 app on your device.
2. **Select a Typing Test**: Click on the "Start Typing" button to begin a typing test.
3. **Type the Prompt**: Follow the on-screen text prompt and type as accurately and quickly as possible.
4. **View Results**: After completing the test, view your WPM and accuracy results.
5. **Try Again**: Click the "Try Again" button to retake the test or select a new prompt.
